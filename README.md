# Proyecto de Portafolio Web Progresivo

Este repositorio contiene el código de un portafolio web personal, desarrollado progresivamente a través de 3 Historias de Usuario (HU). Está pensado como un proyecto educativo para aprender HTML5, CSS3 y JavaScript básico, aplicando buenas prácticas de desarrollo y arquitectura frontend.

## 🚀 Tecnologías Usadas
- **HTML5:** Marcado semántico para mejorar la accesibilidad y el SEO.
- **CSS3:** Estilos en cascada modernos, uso de Flexbox, CSS Grid, Media Queries y variables CSS.
- **JavaScript (Vanilla JS):** Interactividad básica, manipulación del DOM y eventos.
- **Git y GitHub:** Control de versiones.

## 📁 Estructura del Proyecto

El proyecto está dividido en carpetas que representan la evolución del código a través de diferentes etapas de desarrollo:

```text
hu_html/
│
├── README.md                  # Documentación del proyecto y guía de ramas
│
├── hu1-portafolio/            # HU1: Estructura básica HTML y CSS inicial
│   ├── index.html
│   ├── mascotas.html
│   └── styles.css
│
├── hu2-layout/                # HU2: Mejoras de diseño usando Flexbox y CSS Grid
│   ├── index.html
│   ├── mascotas.html
│   └── styles.css
│
└── hu3-responsive-js/         # HU3: Diseño UI Resposive (Móvil/Tablet/Desktop) y JavaScript
    ├── index.html
    ├── mascotas.html
    ├── styles.css
    ├── style.min.css          # Simulación de archivo CSS minificado
    └── script.js
```

## 💻 Cómo ejecutar el proyecto
1. Clona o descarga este repositorio en tu máquina local.
2. Abre la carpeta del proyecto en tu editor de código preferido (recomendado **Visual Studio Code**).
3. Instala y utiliza la extensión **Live Server** en VS Code.
4. Navega a cualquiera de las carpetas de las historias de usuario (ej. `hu3-responsive-js`).
5. Haz clic derecho sobre el archivo `index.html` y selecciona **"Open with Live Server"**.
6. Se abrirá una pestaña en tu navegador predeterminado mostrando el proyecto en funcionamiento.

*Este es un proyecto con fines educativos orientado a buenas prácticas en el ecosistema Frontend.*
